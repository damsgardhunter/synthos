Layer 4 — Inverse Learning System

The inverse engine must learn from results.

Every candidate evaluated returns:

Tc
lambda
DOS
stability

This is the training signal.

Implementation Tasks

Create module:

server/inverse/inverse-learning.ts
Step 1 — Compute distance from target

Example metric:

score = 
|Tc_target - Tc| * weight +
|lambda_target - lambda| +
hullPenalty

Example:

target Tc = 290
predicted Tc = 110

distance = 180
Step 2 — Reward near-solutions

Instead of binary success/failure, reward closeness.

Example:

reward = exp(-distance / scale)

This allows gradual improvement.

Step 3 — Update generator biases

Example:

Nb-B-H systems performed well
increase probability of B-H systems

Store:

element success matrix
structure success matrix
