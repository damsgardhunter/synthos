T027 — Atomic Volume Estimator

File:

structure-validator.ts

Add:

function estimateVolumePerAtom(elements, stoich) {
  let total = 0
  let atoms = 0

  for (e in stoich) {
    total += atomicVolume[e] * stoich[e]
    atoms += stoich[e]
  }

  return total / atoms
}

Impact:
