DFT Worker Not Executing

Your geometry optimization failures show this command:

cd /tmp/dft_calculations/opt_GdCaH10_1772918196263 && /home/runner/workspace/server/

Notice the problem:

The command stops at the directory path.

There is no executable after it.

A correct command should look like:

cd ... && pw.x -in scf.in > scf.out

or

cd ... && xtb structure.xyz --opt

Your worker is trying to execute:

/home/runner/workspace/server/

which is a directory, not a program.

So the shell fails immediately.

Fix

In qe-worker.ts you probably have something like:

exec(`cd ${dir} && ${process.cwd()}`)

Replace with something like:

exec(`cd ${dir} && pw.x -in scf.in > scf.out`)

or whatever your actual binary is.

This is the main reason geometry optimizations fail.
