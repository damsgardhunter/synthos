Your structure scaling loop is running twice (bug)

You have duplicate validation messages:

[DFT] WSe2N: Structure validation attempt 1 — minDist=2.760Å ...
[DFT] WSe2N: Structure validation attempt 1 — minDist=2.760Å ...

This means the validation function is being called twice.

Possible causes:

Likely bug
generateStructure()
   -> validateStructure()

preRelax()
   -> validateStructure()

So validation runs before AND after generation.

But your log still says:

attempt 1
attempt 1

Meaning the counter resets.

Fix

Create a single validation pipeline:

generateStructure
      ↓
validateStructure
      ↓
repairStructure (scale lattice)
      ↓
validateStructure
      ↓
xtbRelax

And track attempt number globally.

Example:

function validateAndRepair(structure) {
  let attempts = 0

  while (attempts < 3) {
    const result = validateGeometry(structure)

    if (result.valid) return structure

    structure = repairStructure(structure, result)
    attempts++
  }

  throw new Error("Structure unrecoverable")
}
