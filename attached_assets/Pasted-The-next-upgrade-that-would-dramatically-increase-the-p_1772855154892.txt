The 4 Major Pairing Mechanisms
1. Electron-Phonon Pairing
4
Physics

Electrons interact through lattice vibrations.

electron → lattice distortion → second electron attracted

This forms a Cooper pair.

Key equation:

Tc = (ωlog / 1.2) * exp(-1.04(1+λ) / (λ - μ*(1+0.62λ)))

Where

λ = electron-phonon coupling
Already implemented in your system

You already compute:

λ

ωlog

μ*

phonon spectrum

So your platform already models this mechanism extremely well.

Feature to add

Add mode-resolved coupling:

λ_qν

Where

q = phonon momentum
ν = phonon mode

This lets the system detect soft phonon instabilities.

2. Spin-Fluctuation Pairing
4
Physics

Electrons pair through magnetic fluctuations.

Occurs when the material is close to:

antiferromagnetism

Pairing interaction:

V(k,k') ≈ g² χ(q)

Where

χ(q) = spin susceptibility
Found in
Material	Tc
cuprates	135 K
pnictides	55 K
heavy fermion	2–20 K
Feature your system should compute
spinFluctuationStrength

Approximation:

χ(q) ≈ DOS(EF) * nestingScore

Your system already has both inputs.

Implementation

Add to physics engine:

spinCoupling = DOS(EF) * nestingStrength * correlationFactor

Where

correlationFactor = U/W
3. Orbital-Fluctuation Pairing
Physics

Electrons in different orbitals fluctuate:

d_xy ↔ d_xz ↔ d_yz

These fluctuations mediate pairing.

Common in multi-orbital transition metal systems.

Found in
Material	orbitals
Fe pnictides	d orbitals
nickelates	d orbitals
ruthenates	t2g orbitals
Feature to compute
orbitalDegeneracy

Example heuristic:

degeneracy = number_of_partially_filled_d_orbitals

Pairing proxy:

orbitalFluctuation = degeneracy * DOS(EF)
4. Excitonic Pairing
4
Physics

Electron-hole excitations form a bound state:

electron + hole → exciton

This can mediate pairing.

Occurs in systems with:

small band gaps
or semimetallic overlap
Found in
Material	type
Ta2NiSe5	excitonic
layered TMDs	excitonic
graphene heterostructures	predicted
Feature to compute
excitonCoupling

Approximation:

excitonCoupling = DOS(EF) / bandGap

Where

bandGap ≈ small or negative
