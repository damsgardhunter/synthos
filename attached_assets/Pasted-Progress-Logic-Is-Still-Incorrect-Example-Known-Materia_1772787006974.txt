Why this is bad

full dataset ingestion

Fix

Progress should be calculated dynamically.

progress = min(100, (itemsLearned / expectedDatasetSize) * 100)

These are not canonical chemical formulas.

Problems this causes

Correct canonical forms
C1 Nb1 → NbC
C1 Tl1 → TlC
Fe0.07C → Fe0.07C0.93 or FeC13 (depending on model)

You should normalize using:

sortByElectronegativity()
